---
title: "Update Company"
api: "PUT /api/v1/companies/{id}"
---

Update an existing company's information and configuration.

## Authentication

This endpoint requires authentication. Include your API token in the request header:

```bash
Authorization: Bearer YOUR_API_TOKEN
```

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the company to update
</ParamField>

## Request Body

<ParamField path="ruc" type="string" required>
  11-digit tax identification number (RUC). Must be unique in the system.
</ParamField>

<ParamField path="razon_social" type="string" required>
  Legal business name (max 255 characters)
</ParamField>

<ParamField path="nombre_comercial" type="string">
  Commercial or trading name (max 255 characters)
</ParamField>

<ParamField path="direccion" type="string" required>
  Business address (max 255 characters)
</ParamField>

<ParamField path="ubigeo" type="string" required>
  6-digit UBIGEO code for geographic location
</ParamField>

<ParamField path="distrito" type="string" required>
  District name (max 100 characters)
</ParamField>

<ParamField path="provincia" type="string" required>
  Province name (max 100 characters)
</ParamField>

<ParamField path="departamento" type="string" required>
  Department/region name (max 100 characters)
</ParamField>

<ParamField path="telefono" type="string">
  Phone number (max 20 characters)
</ParamField>

<ParamField path="email" type="string" required>
  Company email address (max 255 characters, must be valid email format)
</ParamField>

<ParamField path="web" type="string">
  Company website URL (max 255 characters, must be valid URL)
</ParamField>

<ParamField path="usuario_sol" type="string" required>
  SUNAT SOL username (max 50 characters)
</ParamField>

<ParamField path="clave_sol" type="string" required>
  SUNAT SOL password (max 100 characters)
</ParamField>

<ParamField path="certificado_pem" type="file">
  Digital certificate file (.pem, .crt, .cer, .txt, max 2MB)
</ParamField>

<ParamField path="certificado_password" type="string">
  Password for the digital certificate (max 100 characters)
</ParamField>

<ParamField path="endpoint_beta" type="string">
  Custom SUNAT beta endpoint URL
</ParamField>

<ParamField path="endpoint_produccion" type="string">
  Custom SUNAT production endpoint URL
</ParamField>

<ParamField path="modo_produccion" type="string">
  Environment mode: `true`, `false`, `1`, or `0`
</ParamField>

<ParamField path="logo_path" type="file">
  Company logo image (.png, .jpg, .jpeg, max 2MB)
</ParamField>

<ParamField path="activo" type="boolean">
  Active status
</ParamField>
